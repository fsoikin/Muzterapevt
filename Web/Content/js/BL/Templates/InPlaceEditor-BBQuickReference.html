bold, italic, underline
[b]this is bold[/b]
header & big header
[h]my header[/h]
CSS class
[c=orange]this text is orange[/c]
Here are some available classes:
  • orange
  • blue
  • news
Link
[url=http://apreleva.com]Alisa Apreleva[/url]
Bullet list
[list] [*]one [*]two [*]three [/list]
NOTE: the [*] tag doesn't need to be closed.
Embed a video
[video=http://www.youtube.com/abcde]
A few hints:
  • Just copy&paste the URL of the video.
  • The tag doesn't have to be closed.
  • This only works for YouTube right now, but support for other sites is possible.
Insert raw (unprocessed) HTML markup
[html]This is <b>bold</>, ha.[/html]
Force a line break
Line with a[br]break.
NOTE:
  • Line breaks are already present where you hit Enter. No need to add an extra [br].
  • This tag doesn't need to be closed.
