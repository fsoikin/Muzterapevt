See the Examples page for more details on each of these markup elements.
double asterisk bold
**this is bold**
single asterisk italic
*this is italic*
underscore character underline
_this is underlined_
double or single dash header & big header
--my header--
NOTE: The header must be the only thing on the line.
CSS class
[c=orange]this text is orange[/c]
Here are some available classes:
  • orange
  • blue
  • news
Link
[url=http://apreleva.com]Alisa Apreleva[/url]
space+asterisk Bullet list

* one
* two
* three
NOTE: each asterisk should be on a new line, and there should be at least one space in front of each asterisk.
Embed a video
[video=http://www.youtube.com/abcde]
A few hints:
  • Just copy&paste the URL of the video.
  • This only works for YouTube right now, but support for other sites is possible.
Insert raw (unprocessed) HTML markup
[html]This is <b>bold</>, ha.[/html]
Insert attached picture
[img=my_picture.jpg]
You must first attach the picture to the page by dragging&dropping it on the text editor.
Insert a link to an attached file
[file=brochure.jpg]Download brochure[/file]
You must first attach the file to the page by dragging&dropping it on the text editor.
